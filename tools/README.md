# 레가이아 전설 한글 번역 툴

PS1 Legend of Legaia (USA) 한글 패치 제작 툴.

## 준비

**필요한 것**
- Python 3.8 이상
- 원본 게임 디스크 이미지 (`Legend of Legaia (USA).bin`)

**설치**
```
pip install pillow numpy
```

Windows 는 Python 에 tkinter 가 포함되어 있어 추가 설치가 필요 없습니다.
(Linux 는 `sudo apt install python3-tk`)

## 1) 원본에서 PROT.DAT / EXE 뽑기

```
python3 tools/extract.py "Legend of Legaia (USA).bin" out/
```
→ `out/PROT.DAT`, `out/SCUS_942.54` 생성

## 2) 번역 툴 실행

```
python3 tools/translator_gui.py
```

**사용법**
1. **[PROT.DAT 열기]** → 전체 대사 추출 후 **맵별로 분할 저장** (build/maps/)
2. 왼쪽에서 맵을 고르고, 가운데 목록에서 대사를 클릭
3. 아래 편집기에 번역 입력 → **Ctrl+Enter** 로 적용
   → **그 맵 파일만 즉시 저장** (작고 안전, 자동 백업)
4. **[검색]** 으로 영어/한글 대사를 빨리 찾아 수정
   - 전체 맵 / 현재 맵 범위 선택, 미번역만 필터
5. **[빌드]** → 맵별 파일을 합쳐 PROT.DAT + EXE 생성

### 왜 맵별 저장인가
예전엔 text_dump.json(3MB) 하나에 통째로 저장해서, 파일이 커서
저장이 실패하고 번역이 날아갔다. 이제 맵별(build/maps/map6.json 등)로
나눠 각 파일이 작고, 편집 즉시 그 맵만 저장되어 안전하다.
저장은 임시 파일에 쓴 뒤 교체하므로 저장 중 중단돼도 원본이 남는다.

## 3) 디스크에 써 넣기

```
python3 tools/write_disc.py 원본.bin out/PROT.DAT out/SCUS_942.54 한글판.bin
```
→ `한글판.bin` + `한글판.cue` 생성. 에뮬레이터로 실행하면 됩니다.

## 번역할 때 지켜야 할 것

### 🔴 매크로는 지우면 안 됩니다
`{NAME}`, `{ITEM}` 등은 캐릭터/아이템 이름을 넣는 **제어 코드**입니다.
위치는 옮겨도 되지만 **지우면 게임이 멈춥니다.**

```
원문:  Look, {NAME}. That is
번역:  봐라 {NAME}. 저것이     ← {NAME} 유지 ✓
번역:  봐라 저것이             ← 게임 멈춤 ✗
```

툴이 자동으로 검사해서 **주황색**으로 표시하고, 빌드를 막습니다.

### 길이 제한
번역은 **원문 바이트 수 이하**여야 합니다. 텍스트가 스크립트 코드 안에
박혀 있어서 길이를 늘리면 게임이 깨집니다.

- 한글 1글자 = **2바이트**
- 영문/기호/공백 = 1바이트
- 매크로 = 2바이트

편집기 오른쪽에 `현재/한계` 가 실시간으로 뜹니다.
- 🟢 초록 = 여유 있음
- 🟠 주황 = 빠듯함 (페이지 전환 2B 가 붙으면 넘칠 수 있음)
- 🔴 빨강 = 초과 (삽입 안 됨)

### 음절 400자 제한
폰트가 VRAM 에 400칸뿐입니다. 상단 막대에 사용량이 표시됩니다.
넘으면 일부 대사가 안 나옵니다.

실측상 한 지역(맵) 기준으로는 여유가 있지만, 전체 번역 시에는
자주 쓰는 음절 위주로 관리해야 할 수 있습니다.

### 슬롯이 꽉 찬 맵
일부 맵은 원본 데이터가 슬롯을 거의 다 채워서(여유 7~11B) 한글이
들어가지 않습니다. 빌드 로그에 경고가 뜨고, 그 맵은 영어로 남습니다.

## 툴 구성

| 파일 | 역할 |
|---|---|
| `translator_gui.py` | **번역 GUI** |
| `translate.py` | 번역 파이프라인 (CLI: `--analyze` / `--build`) |
| `dump_text.py` | 대사 추출 |
| `write_disc.py` | 디스크 이미지 생성 |
| `hook7.py` | 한글 렌더링 훅 (MIPS 어셈블리) |
| `protected.py` | 보호구역 검증 |
| `legaia.py` | PROT/PACK/LZSS |
| `Galmuri11.ttf` | 한글 픽셀폰트 |

## 안전장치

빌드 시 다음을 자동 검사하고, 걸리면 **중단**합니다.

| 검사 | 막는 것 |
|---|---|
| 보호구역 | 숫자 폰트·팔레트 파괴 (메뉴 숫자가 사라짐) |
| 매크로 보존 | 제어코드 삭제 (게임 멈춤) |
| 왕복 검증 | 인코딩 버그 (글자 깨짐) |
| 길이 검사 | 원문보다 긴 번역 |

## 속도

빌드는 보통 **수 초**입니다. 압축 결과를 캐시하므로 두 번째부터 더 빠릅니다.

`tools/lzss_fast` (C 구현)가 있으면 더 빠릅니다:
```
gcc -O2 -o tools/lzss_fast tools/lzss_fast.c
```
없어도 파이썬 구현으로 동작합니다 (출력 동일).
